# 사용 예시

ku-portal-mcp의 다양한 활용 예시를 소개합니다.

## 내 수강신청 내역 조회

학수번호, 강의시간, 강의실, 교수 등 상세 정보를 확인합니다.

```
> 내 수강과목 보여줘
```

```json
{
  "success": true,
  "year": "2026",
  "semester": "1",
  "total_credits": "6.0",
  "count": 3,
  "courses": [
    {
      "course_code": "AAI110",
      "section": "00",
      "course_type": "전공선택",
      "course_name": "딥러닝",
      "professor": "석흥일",
      "credits": "2(2)",
      "schedule": "월(7-8) 애기능생활관 301호",
      "retake": false,
      "status": "신청"
    }
  ]
}
```

```
> 총 몇 학점 신청했어?
→ 총 6.0학점 신청하셨습니다.

> 내 수업 강의실 어디야?
→ 딥러닝: 월(7-8) 애기능생활관 301호
→ 자연어처리: 수(7-8) 우정정보관 202호
→ 텍스트마이닝: 목(7-8) 우정정보관 202호
```

## 공지사항 조회 + 검색

```
> 최근 공지사항 보여줘
> "수강신청" 관련 공지 검색해줘
> 공지사항 3번째 글 상세 내용 보여줘
```

## 도서관 좌석 현황

```
> 중앙도서관 빈자리 몇 개야?
> 전체 도서관 좌석 현황 알려줘
```

## 수업시간표

```
> 이번 주 시간표 보여줘
> 월요일 수업 뭐 있어?
> 시간표를 ICS 파일로 만들어줘
```

## 개설과목 검색

```
> 정보대학 컴퓨터학과 개설과목 보여줘
> COSE101 강의계획서 보여줘
```

## Canvas LMS

```
> LMS에 어떤 과목 듣고 있어?
> 딥러닝 과제 목록 보여줘
> 아직 안 낸 과제 있어?
> 이번 주 제출할 과제 뭐야?
> 딥러닝 성적 어때?
> 퀴즈 일정 있어?
```
