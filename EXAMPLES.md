# 사용 예시

ku-portal-mcp를 설치한 후 Claude Code에서 아래와 같이 자연어로 물어보세요.

---

## 공지사항

```
> 최근 공지사항 보여줘
> 공지사항 5개만 보여줘
> 공지사항 3번째 글 상세 내용 보여줘
> "수강신청" 관련 공지 검색해줘
> 등록금 관련 공지 있어?
```

## 학사일정

```
> 이번 달 학사일정 알려줘
> 수강신청 일정 언제야?
> 3월 학사일정 보여줘
> 기말고사 일정 검색해줘
```

## 장학공지

```
> 장학금 관련 공지 있어?
> 최근 장학 공지 보여줘
> 국가장학금 관련 공지 찾아줘
```

## 통합 검색

```
> "수강신청" 관련해서 공지, 일정, 장학 전부 검색해줘
> "졸업" 키워드로 통합 검색
> "등록금" 관련 정보 전부 찾아줘
```

## 도서관 좌석 (로그인 불필요)

```
> 도서관 빈자리 있어?
> 중앙도서관 좌석 현황 보여줘
> 과학도서관 빈자리 몇 개야?
> 전체 도서관 좌석 현황 알려줘
> 노트북 사용 가능한 열람실 어디야?
> 지금 가장 자리 많은 도서관 어디야?
```

## 수업시간표

```
> 내 시간표 보여줘
> 이번 주 시간표 알려줘
> 월요일 수업 뭐 있어?
> 오늘 수업 몇 시에 시작해?
> 시간표를 ICS 파일로 만들어줘
> 내일 공강이야?
```

## 개설과목 / 강의계획서

```
> 정보대학 컴퓨터학과 개설과목 보여줘
> 이번 학기 경영대학에 어떤 학과가 있어?
> 공과대학 개설과목 검색해줘
> COSE101 강의계획서 보여줘
> 인공지능 관련 과목 있어?
```

## Canvas LMS — 수강과목

```
> LMS에 어떤 과목 듣고 있어?
> 이번 학기 수강 과목 목록 보여줘
```

## Canvas LMS — 과제

```
> 딥러닝 과제 목록 보여줘
> 아직 안 낸 과제 있어?
> 이번 주 제출할 과제 뭐야?
> 과제 마감일 임박한 거 알려줘
```

## Canvas LMS — 강의자료

```
> 자연어처리 강의자료 보여줘
> 1주차 강의 자료 뭐 있어?
> 이번 주 올라온 강의 자료 있어?
```

## Canvas LMS — 대시보드 / 할 일

```
> LMS 대시보드 보여줘
> 할 일 목록 알려줘
> 다가오는 이벤트 있어?
```

## Canvas LMS — 성적

```
> 딥러닝 성적 어때?
> 전체 과목 성적 보여줘
> 현재 학점 어떻게 돼?
```

## Canvas LMS — 제출 현황

```
> 과제 제출 현황 보여줘
> 딥러닝 과제 제출했어?
> 미제출 과제 있어?
> 지각 제출한 과제 있어?
```

## Canvas LMS — 퀴즈

```
> 퀴즈 일정 있어?
> 이번 주 시험 있어?
> 딥러닝 퀴즈 목록 보여줘
```

---

## 조합해서 물어보기

여러 기능을 한번에 활용할 수도 있습니다:

```
> 오늘 수업 끝나고 도서관 갈 건데, 빈자리 있는 곳 알려줘
> 이번 주 과제 마감이랑 시험 일정 정리해줘
> 내일 수업 뭐 있고, 제출할 과제 있어?
> 수강신청 기간이랑 관련 공지 같이 알려줘
```
